# Excel Add-In

- Generates web form from excel

## Development Environment

- Windows 10, 11
- Excel 2021+
- npm version 10+
- Change directory to `/excel-add-in`
- Setup node packages `npm install`
- Run `npm run build`

- VS Code
  - `RUN AND DEBUG` tab
  - Select `Excel Desktop (Edge Chromium)`
  - Run

## Deploying

- Run `npm run build`
- Copy `manifext.xml` and `dist/`
